    First start

    Upon opening the program you will be greeted with the interface that consists of: a menu, indicator text, "Input configuration" panel, "Options" panel and "MIDI file player" panel. You must have at least one MIDI input device connected for the MIDI input to keystrokes feature to work. If you have none connected you can still use the MIDI file to keystrokes feature.

    Note: a MIDI input device here is most commonly a MIDI keyboard/electronic piano (though it can be any other) that can connect to a PC via USB or any other method and transmit MIDI messages.

    So, if you have your device connected you need to select it in the "Device" combobox in the "Input configuration" panel, if it hasn't been already selected. If you don't see your device in there that might mean that it is busy already being used by another program. If you are sure that it is connected and free - please message the author about this error. If your device wasn't connected before the start of the program but is now - click the "Update" button to see it in the combobox.

    The indicator text will display the last MIDI input note details, such as key, velocity, the corresponding keystroke and whether it was registered or not.

    Now everything basic needed for the MIDI input to keystrokes feature is configured and you can start playing. Look below for the advanced options.
